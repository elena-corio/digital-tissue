## Getting Started

This is a Vue.js frontend application with a structured folder organization for scalable development.

### Folder Structure

- **config/**: Configuration files including UI text constants
- **components/**: Reusable Vue components organized by type
  - **ui/**: Generic UI components
  - **buttons/**: Button components
  - **cards/**: Card components
  - **layout/**: Layout components
  - **navigation/**: Navigation components
- **views/**: Page-level Vue components
- **router/**: Vue Router configuration
- **assets/styles/**: Global CSS stylesheets
  - **colors.css**: Color variables
  - **typography.css**: Typography styles
  - **globals.css**: Global styles and imports
